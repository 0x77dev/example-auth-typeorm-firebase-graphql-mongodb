# Sample Backend with Firebase Auth

## Requirements

- [x] TypeORM

### Auth methods

- [ ] Email & Password
- [ ] Google

## Project setup

### Installing deps

```bash
yarn install
```

### Starting in dev/watch mode

```bash
yarn start:dev
```

### Building for production

```bash
yarn build
```

### Starting for production

```bash
export NODE_ENV=production

yarn start
```
